# 💰 Simple Bank Application (HTML + JavaScript)

## 📌 Project Description

This project is a basic **Bank Application** built using HTML and JavaScript.
It allows users to **deposit money, withdraw money, and check balance**.

---

## 🛠 Technologies Used

* HTML
* JavaScript

---

## ⚙️ Features

* Deposit money
* Withdraw money
* Check current balance
* Prevents withdrawal when balance is low
* Simple and user-friendly interface

---

## 🧠 Logic Used

* A variable `balance` is used to store current money
* User enters amount using input field
* JavaScript functions perform operations:

  * `deposit()` → adds money
  * `withdraw()` → subtracts money
